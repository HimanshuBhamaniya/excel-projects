# Excel Projects Repository

This repository is a collection of my Excel-based projects.  
Each project is organized into its own folder and contains:

- **Excel workbook(s)** with the main analysis or model
- **Images** for charts, dashboards, or visual outputs
- **README.md** with project-specific documentation

## Purpose
The goal of this repository is to:
- Keep all Excel projects in one place
- Maintain a clear structure for multiple projects
- Provide easy navigation and reference for future work

│
├── excel_project_2/

## How to Use
- Navigate to the project folder you’re interested in.
- Open the Excel file to explore the analysis.
- Refer to the project’s README.md for details on methodology, data, and insights.
